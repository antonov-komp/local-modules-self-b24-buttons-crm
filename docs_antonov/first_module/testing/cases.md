## Тест-кейсы для модуля «BP Button Field»

Здесь будут описаны конкретные сценарии тестирования:

- Создание поля `bp_button_field` и проверка появления в админском реестре.
- Нажатие на кнопку в разных сущностях CRM (лид, сделка, контакт, компания).
- Поведение при отсутствии прав, ошибках обработчика, проблемах с сетью.
- Тесты будущего функционала (условия отображения, интеграция с роботами, логирование).

Каждый кейс будет оформлен с шагами, ожидаемым результатом и примечаниями.
