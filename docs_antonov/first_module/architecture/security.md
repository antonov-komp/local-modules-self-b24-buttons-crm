## Безопасность модуля «BP Button Field»

Здесь фиксируются требования и решения по безопасности:

- Ограничение размещения кода в `/local/modules/`.
- Использование `check_bitrix_sessid()` для AJAX и форм.
- Проверка прав доступа через `CCrmAuthorizationHelper` и другие механизмы.
- Обработка ошибок и логирование без утечек чувствительной информации.

Конкретные реализации будут увязаны с контроллерами и событиями, описанными в `backend_d7.md`.
