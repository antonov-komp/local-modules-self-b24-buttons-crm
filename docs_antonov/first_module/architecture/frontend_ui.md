## Архитектура frontend/UI модуля «BP Button Field»

Документ описывает:

- Поведение кнопки в карточках CRM и смарт-процессов.
- JS-расширения модуля (на базе Bitrix Core JS).
- Механику работы с `BX.SidePanel`, iframe и передачей контекста (`ENTITY_ID`, `ELEMENT_ID`, `FIELD_ID`, `USER_ID`).
- Используемые UI-библиотеки (`ui.buttons`, `ui.sidepanel`, `ui.notification`).

Детализация будет выполняться по мере проработки сценариев.
