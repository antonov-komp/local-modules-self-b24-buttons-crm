## Модуль «BP Button Field» (коробка Битрикс24)

Этот модуль расширяет стандартные возможности CRM Битрикс24, добавляя новый тип пользовательского поля — «Кнопка управления логикой» (`bp_button_field`).

### Назначение документа

- Дать краткий обзор целей и ключевого функционала модуля.
- Зафиксировать структуру документации по модулю внутри `docs_antonov/first_module/`.

### Структура документации модуля

```text
first_module/
├── overview.md               # Обзор и цели
├── requirements.md           # Требования
├── architecture/             # Архитектура
├── module_structure/         # Структура модульных файлов
├── tasks/                    # TASK-файлы для разработки
├── api/                      # Документация API модуля
├── ui_ux/                    # UX/UI спецификации
├── testing/                  # Тестирование
└── local_button.md           # Базовая спецификация (master-документ)
```
