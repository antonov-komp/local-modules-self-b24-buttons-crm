## Модуль «BP Button Field» (коробка Битрикс24)

Модуль «BP Button Field» расширяет стандартные возможности CRM Битрикс24 **только для коробочной версии продукта**.  
Он добавляет новый тип пользовательского поля — «Кнопка управления логикой» (`bp_button_field`), который позволяет:

- отображать в карточках CRM и смарт‑процессов интерактивные кнопки вместо статичных значений;
- запускать бизнес‑логику без перезагрузки страницы (через `BX.SidePanel` / iframe);
- централизованно настраивать поведение кнопок через административный реестр.

### Основные сценарии использования

- **Администратор CRM**:
  - создаёт пользовательское поле типа `bp_button_field` для нужных сущностей (лиды, сделки, контакты, компании, смарт‑процессы);
  - в административном реестре модуля настраивает для каждой кнопки URL обработчика, заголовок и ширину слайдера.
- **Оператор/менеджер по продажам**:
  - видит в карточке CRM кнопку вместо обычного поля;
  - нажимает её и получает SidePanel/iframe с нужным интерфейсом (формой, отчётом, внешним сервисом и т.п.), без выхода из карточки.

### Ограничения версии

- Целевая платформа: **1С‑Битрикс24, коробочная редакция** (подразумевается доступ к `/local/` и D7‑ORM).
- Облачная версия Битрикс24 **не рассматривается** в рамках данного модуля.
- Все пути и примеры кода привязаны к размещению модуля в `/local/modules/my.bpbutton/`.
